# ♟️ Ultimate Chess Pro – Master Edition

**Ultimate Chess Pro – Master Edition** is a fully functional, browser-based chess game built using **HTML, CSS, and Vanilla JavaScript**.
It delivers a **modern UI**, **complete chess rules**, and an **immersive two-player experience** without requiring any external libraries or frameworks.

## 🚀 Features

### 🎮 Gameplay
* Two-player **local chess gameplay**
* **Turn-based system** (White vs Black)
* Full implementation of **standard chess rules**
* **Check**, **Checkmate**, and **Stalemate** detection
* **Pawn promotion** with user selection
* **Castling** (King-side & Queen-side) with rule validation

### 🎨 User Interface
* Modern **dark-themed UI**
* Responsive **8×8 chessboard**
* Fully white and fully black chess pieces
* Smooth animations and visual effects
* **Red highlight & pulse animation** when king is in check
* End-game **result overlay screen**

### 🧠 Game Logic
* Move validation for all pieces
* Illegal move prevention
* King safety enforcement
* No external libraries used
* Written entirely in **pure JavaScript**

## 🛠️ Technologies Used
* **HTML5** – Structure
* **CSS3** – Styling, animations, responsiveness
* **JavaScript (ES6)** – Game logic and interactions

## ▶️ How to Run
1. Download or clone this repository
2. Open the file `index.html` in any modern web browser
3. Start playing instantly — **no setup required**

## 🎯 Rules Supported
* ✔ Pawn movement (single, double step, capture)
* ✔ Rook, Bishop, Knight, Queen, King movements
* ✔ Castling restrictions (no check, no obstruction)
* ✔ Pawn promotion (Queen, Rook, Bishop, Knight)
* ✔ Check & Checkmate detection
* ✔ Stalemate detection

## 🧩 Highlights
* No legal-move hints shown (forces real chess thinking)
* Visual indication when a king is under attack
* Game locks automatically after checkmate or draw
* Restart option via **Play Again** button


## 📱 Responsiveness
* Works on **desktop**, **tablet**, and **mobile browsers**
* Board scales automatically based on screen size
